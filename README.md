# Option → Stock Move Calculator

Upload an option chain screenshot, extract Greeks, and calculate
the stock price move required for an option to reach a target price.

## How to use
1. Upload screenshot
2. Run OCR (optional helper)
3. Verify/edit Greeks manually
4. Enter target option price
5. Choose time horizon
6. Click calculate

## Notes
- Uses delta + gamma + theta approximation
- Best for 0–2 DTE options
- Recalculate after large price moves
